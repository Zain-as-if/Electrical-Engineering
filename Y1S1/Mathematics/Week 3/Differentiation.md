# Basic Rules

$$\begin{align*}
\frac{d}{dx}(x^n)&=nx^{n-1}\\
\frac{d}{dx}(e^x)&=e^x\\
\frac{d}{dx}(\log(ax+b))&=\frac{\frac{d}{dx}(ax+b)}{ax+b}\\
\frac{d}{dx}(\ln(x))&=\frac{1}{x}\\
\frac{d}{dx}\sin(x)&=\cos(x)\\
\frac{d}{dx}\cos(x)&=-\sin(x)\\
\frac{d}{dx}\sinh(x)&=\cosh(x)\\
\frac{d}{dx}\cosh(x)&=\sinh(x)
\end{align*}$$

# Combination Of Rules
## Sum
$$\frac{d}{dx}(f+g)=\frac{df}{dx}+\frac{dg}{dx}$$
## Product Rule
$$\frac{d}{dx}(f\cdot g)=(\frac{df}{dx})g+(\frac{dg}{dx})f$$

## Quotient Rule
$$\frac{d}{dx}(\frac{f}{g})=\frac{1}{g^2}[(\frac{df}{dx})g - (\frac{dg}{dx})f]$$
![[Pasted image 20251230021357.png]]
## Chain Rule
$$\frac{d}{dx}[f(u(x))]=\frac{df}{du} \cdot \frac{du}{dx}$$![[Pasted image 20251230021413.png]]

# Increasing / Decreasing Functions

Function is increasing when $\frac{d}{dx}>0$
Function is decreasing when $\frac{d}{dx}<0$

If you want to find whether function is increasing or decreasing steps are:
1. Find derivative
2. Derivative = 0, Solve
3. Figure out ranges
