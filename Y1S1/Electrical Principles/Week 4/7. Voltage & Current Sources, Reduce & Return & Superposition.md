# Voltage Sources

All voltage sources have internal resistance that can cause unexpected results at terminals

## Ideal Voltage Source

Constant voltage regardless of current
At every level of current the terminal voltage of the battery is still E volts

For this to happen the voltage source must have an internal resistance which is very low compared to the resistance of the load it is powering

# Source Conversions

Sources can be converted from voltage to current and vice versa
- Internal resistance must be present (can only do it on 'real' source)
- These are only equivalent at the external terminals

Objective of conversion is for the load to receive the same voltage, current and power from each source

If achieved, load doesn't know or care whether a voltage or current source is present

- Make sure to keep current flow in the same direction (+ve terminal -> -ve terminal)
- Check voltage is the same

Usually circuits contain both series and parallel elements
Use reduce and return methods

To find unknown voltage:
1. Reduce - Identify elements that can be combined in series/parallel, redraw circuit and calculate total resistance
2. Calculate total current
3. Return - Substitute back and calculate the voltage

## Principle Of Superposition

If $x_{i}(t)\mapsto y_{i}(t)$
then $x_{1}(t)+x_{2}(t)\mapsto y_{1}(t)+y_{2}(t)$

If an input $x_{1}(t)$ results in output $y_{1}(t)$
and an input $x_{2}(t)$ results in output $y_{2}(t)$
then an input $x_{1}(t)+x_{2}(t)$ will result in an output $y_{1}(t)+y_{2}(t)$

# Superposition For Circuits

Linear circuits (like all linear systems) obey the principle of superposition

Current through (or voltage across) any element in a circuit with multiple voltages and/or current sources is sum of current through (or voltages across) the same element due to each source alone 

Helps us solve complex circuits with both current and voltage sources

![[Pasted image 20260109032219.png]]
?
## Recipe For Circuits

Recipe for circuits:
1. Consider effect of each source in turn (separately)
2. EMF of voltage sources not being considered should be set to zero (i.e. $E=0\ V$)
3. Voltage sources not being considered should be short-circuited (S/C)
	- Not strictly necessary because ideal voltage source already has zero internal resistance and so is already S/C
4. Current of current sources not being considered should be set to zero (i.e. $I=0\ A$)
5. Current sources not being considered should be open-circuited (O/C)
	- Not strictly necessary because ideal current source already has infinite resistance and so is already O/C
6. Don't ignore (internal) source resistances of non-ideal sources if they are given
7. Calculate current (or voltage) across element(s) of interest
8. Repeat for each source in turn
9. For element(s) of interest sum individual currents (or voltages) found for each source
	- Taking care with directions of currents and polarity of voltages

Superposition states that the output of two or more inputs is the sum of the outputs that would have been caused by each input individually
