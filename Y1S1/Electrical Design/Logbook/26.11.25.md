4 Pin (Power 12V, Ground, Tachometer (Speed Feedback), PWM) PWM fans 
Tachometer pin can tell us actual fan speed, detects fan failure, closed-loop speed control (precise RPM control, how motherboards are able to ensure exact fan speed)
Tachometer pin not needed

https://www.instructables.com/4-Wire-Fan-PWM-Speed-Controller-Using-555-Timer-IC/

4 Pin fans use 12V supply for the fan, w/ voltage regulator to provide the 5 volts needed to produce & output the PWM (Pulse Width Modulation)

Ideally duty cycle at frequency of 25KHz, in practice anything between 
15-40KHz fine (Will vary between fans, testing required) 

Can use Arduino Nano to provide PWM, already has voltage regulator needed 

Suggest temp ranges:
- 0-10°C - Fan Off - 0% Duty Cycle
- 10-20°C - Fan Slow - 20% Duty Cycle
- 20-30°C - Fan Medium (Pc Average Idle Speed) - 50% Duty Cycle
- 30-40°C - Fan High - 100% Duty Cycle

Add ~0.5–1.0°C hysteresis on each threshold so no rapid toggling

<hr>
Hysteresis:

If temps go above 50C -> ramp up fans.

If temps go under 48C -> ramp down fans.

This helps in case the temps reach 50C and stay around that value (varies between 49.8-50.1). Cause without hysteresis this would cause the fans to constantly ramp up and down. But with hysteresis it makes sure that only ramps down once its cool enough so it wont ramp up again in 2 seconds.
<hr>

Don't use push-pull for the fan, use open-drain instead
push-pull actively drives signal both high & low
open-drain can only drive signal low, requires external pull-up resistor to drive it high

*Many fans tolerate push-pull at 5V, however expect open-drain*

Don't connect Fan +12V to Arduino, only go to the 12V rail

<h1>Alternatives</h1>
- LEDs - different colour LEDs all connected, based on different temperature ranges different ones turn on, coloured LEDs would work best 
- Using multimeter with frequency measurement on MOSFET gate/drain & oscilloscope to view duty cycle and waveform shape (look for clean ~25KHz (or whatever frequency using) square wave on the gate, inverted pulses on drain)
